# HR-Attrition
HR Analytics EDA project analyzing 1,470 employee records to identify key drivers of attrition. Uncovered insights on salary, overtime, and department trends, and identified high-risk employees. Built using Python, Pandas, NumPy, Matplotlib, and Seaborn for data cleaning and visualization.

📊 HR Analytics – Employee Attrition Analysis

This project focuses on Exploratory Data Analysis (EDA) of an HR dataset containing 1,470 employee records to identify key factors influencing employee attrition.

🔍 Key Insights:
- Overall attrition rate: 16.12%
- Low salary employees are 4x more likely to leave
- Overtime employees are 3x more likely to leave
- Sales department has the highest attrition (20.6%)
- Identified 86 high-risk employees with 43% attrition rate

🛠 Tech Stack:
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

📈 Objectives:
- Understand employee behavior and attrition patterns
- Derive actionable business insights
- Apply EDA techniques in a structured way

This project demonstrates data cleaning, visualization, and insight generation using real-world HR data.
